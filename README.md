vim-cfg
=======

My vim config.

If you want to use it:

    git clone https://github.com/saksmt/vim-cfg.git ~/.vim
    cd ~/.vim/
    git submodule init
    git submodule update --recursive
    ln -s ~/.vim/vimrc ~/.vimrc
